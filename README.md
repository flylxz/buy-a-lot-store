### About The Project

Store with filters

### Built With

- ReactJS
- Material UI
- JSON Server

### Installation

Clone the repo

```sh
git clone https://github.com/flylxz/buy-a-lot-store.git
```

Install NPM packages

```sh
npm install
```

start json server

```sh
npm run serve
```

start app

```sh
npm start
```
